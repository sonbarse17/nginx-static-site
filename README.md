# Nginx Static Site

Static website served by Nginx web server.

## Features
- Nginx web server
- Static content serving
- High performance
- Production ready

## Quick Start
```bash
docker build -t nginx-static-site .
docker run -p 80:80 nginx-static-site
```

Visit: http://localhost
